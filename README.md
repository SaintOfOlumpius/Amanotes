Notes Application

NotsiMate is a new and innovative notes application designed to support both students and working professionals in managing their tasks, ideas, and information effectively.
The app combines simplicity with powerful features, offering a seamless way to create, organize, and access notes anytime, anywhere.
This notes app, is a modern application built with Kotlin in Android Studio, designed with both reliability and convenience in mind. It leverages an offline-first storage approach using Room Database for seamless local storage and automatically synchronizes notes with the cloud once the device is back online. With secure Google Single Sign-On (SSO) integration, users can log in effortlessly while keeping their information protected, making NotsiMate a powerful yet user-friendly solution for managing notes across study, work, and personal life.
